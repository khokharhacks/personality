Quick reference — the three scenarios

SCENARIO 1
Sign only
Email → Hash (SHA‑1) → Sign with PRₐ → Compress
✓ Authentication
✓ Non‑repudiation
✗ Confidentiality
SCENARIO 2
Encrypt only
Email → Compress → Encrypt w/ Kₛ → Encrypt Kₛ with PUᵦ
✗ Authentication
✗ Non‑repudiation
✓ Confidentiality
SCENARIO 3
Sign + Encrypt
Sign (PRₐ) → attach → Compress → Encrypt w/ Kₛ → Encrypt Kₛ with PUᵦ
✓ Authentication
✓ Non‑repudiation
✓ Confidentiality
